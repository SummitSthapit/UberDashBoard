## Overview
This project is an interactive Power BI dashboard built to analyze Uber trip data and uncover insights related to ride demand, booking trends, trip distance, peak hours, revenue patterns, and customer behavior.

The dashboard was designed to transform raw transportation data into meaningful visual insights that can support operational decisions, business optimization, and trend analysis.

## Dashboard Features
- Interactive filters and slicers
- KPI cards for quick insights
- Trip trend analysis
- Revenue and booking analytics
- Peak hour and demand visualization
- Distance and fare analysis
- Dynamic visualizations using Power BI
- Data cleaning and transformation using Power Query
- DAX measures for advanced calculations

## Tools & Technologies Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- CSV Dataset
- Data Visualization Techniques
